# 📊 Enterprise Sales Analytics Dashboard

> **An end-to-end Business Intelligence project built with SQL Server &
> Power BI**

## 🚀 Project Overview

This project showcases a complete Business Intelligence workflow---from
extracting data using SQL Server to building an interactive Power BI
dashboard for business decision-making.

## ✨ Features

-   📈 Executive Sales Dashboard
-   💰 Revenue, Profit & Orders KPIs
-   👥 Customer Performance Analysis
-   🛍️ Product Insights
-   🌍 Regional Sales Analysis
-   ⚡ Dynamic DAX Measures

## 🛠️ Tech Stack

-   SQL Server
-   T-SQL
-   Power BI
-   Power Query
-   DAX

## 📂 Workflow

``` text
SQL Server → Power Query → Data Model → DAX → Interactive Dashboard
```

## 📊 Dashboard Highlights

-   KPI Cards
-   Sales Trends
-   Customer Analysis
-   Product Analysis
-   Geographic Insights
-   Drill-through Reports
